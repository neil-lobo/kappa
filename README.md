# Kappa

> [!WARNING]
> This library is unstable and may break at any time. Stable release TBD

Kappa is a util library for [Chatterino](https://github.com/Chatterino/chatterino2/) plugin development in TS. Using this library directly in lua is not officially supported but can be used if built manually or downloaded from [npm](https://www.npmjs.com/package/@neil-lobo/kappa) directly.

## Install

```
npm install @neil-lobo/kappa
```

## Features

- Simple `fs` utils
- Custom `setInterval` and `clearInterval` implementations
- ID generation
- Configurable logger
- Schema validation
- Result type
- Settings controller

## Planned
